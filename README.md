# Game:
## This game has 3 command:
### new:
to start new game.
### show:
to see the scores.
### end:
to end the game.

#### sample:
- new
- L
- 3
- 1 L_name1 none L_city1 L_food1
- 2 L_name1 L_sur2 L_city2 L_food2
- 4 L_name3 L_sur2 L_city1 l_food3
- 1  :  20
- 2  :  30
- 4  :  20
- show
- 1  :  20
- 2  :  30
- 4  :  20
